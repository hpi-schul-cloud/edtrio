## Setup
Installing Edtr.io is a fairly straight forward process:

```shell
$ git clone git@github.com:schul-cloud/edtrio.git
$ cd edtrio
$ yarn install
```

After the installation is done you can launch the editor via:

```shell
$ yarn start
```
